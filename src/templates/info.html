Hello and welcome to my sample website. It shows a list of asteroids that are predicted to fly near earth on a given day. It's not like we have enough to worry about, right?

Why did I make this?

I wanted to familiarize myself with the AWS platform and how it uses Docker containers. It's not a fancy website, but this turned in to a good learning opportunity nonetheless.

How did I make this site?

I started out by using the stock, Docker image for Apache. Then, I configured it to serve a website that I created with Python FLASK. Finally, I deployed it on Amazon's Elastic Container Service, because it is newbie friendly.(i.e. cheap) All of the data is being pulled from NASA's Near Earth Object(NEO) API.
